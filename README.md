# Toolbox

A collection of algorithms in python as boiler plates for competitive programming.


### Data Structures
* Clustering
* Heap

### Graphs
* Detecting cycles (through DFS)

### Pathfinding
* Bellman Ford
* Breath First Search (BFS)
* Dijkstra's Shortest path
* Flood fill
* Floyd Warshall
* Johnson's algorithm
* Minimax

### Sorting
* Closest Pair of Points
* Mergesort
